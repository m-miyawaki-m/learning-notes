## 凡例

| 表記 | 説明 |
|------|------|
| `Ctrl` | Windowsの場合 |
| `Cmd` | macOSの場合（Windowsは`Ctrl`に読み替え） |
| `Alt` | Windowsの場合 |
| `Option` | macOSの場合（Windowsは`Alt`に読み替え） |
| `→` | 続けて押す（例: `Ctrl+K` `→` `Ctrl+S` = Ctrl+Kを押した後にCtrl+Sを押す） |

**注**: 以下、Windows表記で統一します（macOSユーザーは`Ctrl`→`Cmd`、`Alt`→`Option`に読み替え）

## 基本操作

### コマンドパレット

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Ctrl+Shift+P` | コマンドパレットを開く | すべてのコマンドにアクセス |
| `Ctrl+P` | ファイルをすばやく開く | ファイル名で検索 |
| `Ctrl+Shift+P` → `@` | シンボルにジャンプ | 現在のファイル内のクラス、関数など |
| `Ctrl+Shift+P` → `#` | ワークスペース内のシンボル | プロジェクト全体から検索 |
| `Ctrl+Shift+P` → `:` | 行番号にジャンプ | 例: `:42` で42行目へ |

### ウィンドウ操作

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Ctrl+Shift+N` | 新しいウィンドウを開く | 別のプロジェクトを開く |
| `Ctrl+W` | ウィンドウを閉じる | 現在のVSCodeウィンドウを閉じる |
| `Ctrl+K` `Ctrl+S` | キーボードショートカットを開く | ショートカット一覧を表示 |
| `Ctrl+,` | 設定を開く | VSCodeの設定画面 |
| `Ctrl+Shift+X` | 拡張機能を開く | 拡張機能のインストール・管理 |

### サイドバー

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Ctrl+B` | サイドバーの表示/非表示 | エディタ領域を広げる |
| `Ctrl+Shift+E` | エクスプローラーを開く | ファイルツリー表示 |
| `Ctrl+Shift+F` | 検索パネルを開く | プロジェクト全体を検索 |
| `Ctrl+Shift+G` | ソース管理を開く | Gitパネル |
| `Ctrl+Shift+D` | デバッグパネルを開く | デバッグビュー |
| `Ctrl+Shift+X` | 拡張機能パネルを開く | 拡張機能の管理 |

## ファイル操作

### ファイル管理

| ショートカット | 機能 | 説明 |
|--------------|------|------|

| `Ctrl+S` | ファイルを保存 | 現在のファイルを保存 |
| `Ctrl+Shift+S` | 名前を付けて保存 | 別名で保存 |
| `Ctrl+K` `S` | すべて保存 | 開いている全ファイルを保存 |
| `Ctrl+W` | タブを閉じる | 現在のエディタタブを閉じる |
| `Ctrl+K` `W` | すべてのタブを閉じる | 開いている全タブを閉じる |
| `Ctrl+Shift+T` | 閉じたタブを再度開く | 最後に閉じたファイルを開く |

### タブ操作

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Ctrl+Tab` | 次のタブへ | タブを順に切り替え |
| `Ctrl+Shift+Tab` | 前のタブへ | タブを逆順に切り替え |
| `Ctrl+1` ~ `Ctrl+9` | N番目のタブへ | 数字でタブを直接選択 |
| `Ctrl+K` `Ctrl+→` | 次のエディタグループへ | 分割エディタ間を移動 |
| `Ctrl+K` `Ctrl+←` | 前のエディタグループへ | 分割エディタ間を移動 |

### エディタ分割

|--------------|------|------|
| `Ctrl+\` | エディタを分割 | 縦に分割 |
| `Ctrl+K` `Ctrl+\` | エディタを横に分割 | 横に分割 |
| `Ctrl+1` / `Ctrl+2` / `Ctrl+3` | エディタグループにフォーカス | 分割したエディタに移動 |
| `Ctrl+K` `←` `→` `↑` `↓` | エディタグループの移動 | タブを別のグループに移動 |

## 編集操作

### 基本編集

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Ctrl+X` | 行を切り取り | 選択範囲がない場合は行全体を切り取り |
| `Ctrl+C` | 行をコピー | 選択範囲がない場合は行全体をコピー |
| `Ctrl+V` | 貼り付け | カーソル位置に貼り付け |
| `Ctrl+Z` | 元に戻す | アンドゥ |
| `Ctrl+Shift+Z` | やり直し | リドゥ |
| `Ctrl+/` | 行コメントの切り替え | `//` または `#` でコメント化 |
| `Shift+Alt+A` | ブロックコメントの切り替え | `/* */` でコメント化 |

### 行操作

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Ctrl+Shift+K` | 行を削除 | カーソル行を削除 |
| `Alt+↑` / `Alt+↓` | 行を上下に移動 | 行を入れ替え |
| `Shift+Alt+↑` / `Shift+Alt+↓` | 行を上下にコピー | 行を複製 |
| `Ctrl+Enter` | 下に行を挿入 | カーソルを移動せず改行 |
| `Ctrl+Shift+Enter` | 上に行を挿入 | カーソルを移動せず上に改行 |
| `Ctrl+]` | インデントを増やす | 選択範囲をインデント |
| `Ctrl+[` | インデントを減らす | 選択範囲のインデントを削除 |
| `Ctrl+Shift+\` | 対応する括弧にジャンプ | `{`, `[`, `(` の対応する括弧へ |

### テキスト選択

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Ctrl+A` | すべて選択 | ファイル全体を選択 |
| `Shift+→` / `Shift+←` | 文字単位で選択 | カーソル移動しながら選択 |
| `Ctrl+Shift+→` / `Ctrl+Shift+←` | 単語単位で選択 | 単語ごとに選択 |
| `Shift+End` / `Shift+Home` | 行末/行頭まで選択 | 行の端まで選択 |
| `Ctrl+L` | 行を選択 | カーソル行全体を選択 |
| `Shift+Alt+→` | 選択範囲を拡大 | 構文に応じて選択範囲を広げる |
| `Shift+Alt+←` | 選択範囲を縮小 | 選択範囲を狭める |

### フォーマット

| ショートカット | 機能 | 説明 |
|--------------|------|------|
| `Shift+Alt+F` | ドキュメント全体をフォーマット | 自動整形 |
| `Ctrl+K` `Ctrl+F` | 選択範囲をフォーマット | 選択部分のみ整形 |
| `Ctrl+Shift+I` | ドキュメント全体をフォーマット | 別の方法 |

## カスタムショートカット設定

### ショートカットのカスタマイズ

VSCodeのショートカットは自由にカスタマイズできます。

#### 設定方法

1. **キーボードショートカット設定を開く**
   ```
   Ctrl+K Ctrl+S
   ```

2. **ショートカットを検索**
   - コマンド名で検索（例: "Format Document"）

3. **ショートカットを変更**
   - コマンド名をクリック
   - `+` アイコンをクリック
   - 新しいキーバインドを入力
   - `Enter` で確定

#### keybindings.json で直接編集

```
Ctrl+Shift+P → "Preferences: Open Keyboard Shortcuts (JSON)"
```

**keybindings.json**:
```json
[
    // ターミナルクリア
    {
        "key": "ctrl+l",
        "command": "workbench.action.terminal.clear",
        "when": "terminalFocus"
    },
    // ファイル保存と整形を同時実行
    {
        "key": "ctrl+s",
        "command": "runCommands",
        "args": {
            "commands": [
                "editor.action.formatDocument",
                "workbench.action.files.save"
            ]
        }
    },
    // Gradleビルドタスクを実行
    {
        "key": "ctrl+shift+b",
        "command": "workbench.action.tasks.runTask",
        "args": "Gradle: Build"
    }
]
```

### 推奨カスタムショートカット

#### 開発効率化

```json
[
    // コンソールログを素早く挿入（JavaScript）
    {
        "key": "ctrl+shift+l",
        "command": "editor.action.insertSnippet",
        "args": {
            "snippet": "console.log('$1', $1)$0"
        },
        "when": "editorTextFocus && editorLangId == 'javascript'"
    },
    // System.out.println を素早く挿入（Java）
    {
        "key": "ctrl+shift+l",
        "command": "editor.action.insertSnippet",
        "args": {
            "snippet": "System.out.println($1);$0"
        },
        "when": "editorTextFocus && editorLangId == 'java'"
    },
    // print() を素早く挿入（Python）
    {
        "key": "ctrl+shift+l",
        "command": "editor.action.insertSnippet",
        "args": {
            "snippet": "print($1)$0"
        },
        "when": "editorTextFocus && editorLangId == 'python'"
    }
]
```
