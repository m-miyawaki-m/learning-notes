# データ構造 学習計画

## 学習目標
- 主要なデータ構造の特性と計算量を理解する
- 適切なデータ構造を選択できるようになる
- Javaの標準ライブラリでの実装を理解する

## 前提知識
- Javaの基本文法
- 配列の基礎
- 計算量（Big O記法）の基礎

## 学習ロードマップ

### Week 1: 線形データ構造
- [ ] 配列とリスト（ArrayList vs LinkedList）
- [ ] スタック（Stack）
- [ ] キュー（Queue, Deque）
- [ ] 各構造の計算量比較

### Week 2: ツリー構造
- [ ] 二分木（Binary Tree）
- [ ] 二分探索木（Binary Search Tree）
- [ ] バランス木（AVL木、赤黒木）
- [ ] TreeMap, TreeSetの内部実装

### Week 3: ハッシュテーブル
- [ ] ハッシュ関数
- [ ] 衝突解決（チェイン法、オープンアドレス法）
- [ ] HashMap, HashSetの内部実装
- [ ] 負荷率とリハッシュ

### Week 4: 応用
- [ ] グラフ（隣接リスト、隣接行列）
- [ ] ヒープ（PriorityQueue）
- [ ] 実務でのデータ構造選択基準
